¿QUÉ SOFTWARE USA {{appName.toUpperCase()}}?

El proyecto de {{appName}} es una re-edición de los ya conocidos tablones de imagenes como 4Chan e Hispachan, implementando nuevas herramientas existentes en el mercado, en este caso se divide en dos partes (front-end y back-end), las herramientas utilizadas para este caso son:

• Angular 8 para el front-end (Typescript)

• NodeJS con Express para el back-end (Typescript).

• MongoDB como base de datos.

• DigitalOcean Space como almacén de archivos estáticos.

¿CÓMO PUEDO COLABORAR?

Aunque {{appName}} funciona con herramientas conocidas y actuales del mercado, aún falta implementar muchos features y si quieres aprender un poco más y aportar código con tu usuario de GitHub, ¡ayudanos!.

Si quieres aportar debes comenzar con copiar el proyecto del repositorio y seguir la documentación para poder correrlo de manera local dentro de tu computador.

Luego ya puedes comenzar con el desarrollo de nuevos features o ideas que tengas para mejorar la plataforma. Todas las peticiónes de código se deben hacer por una "Pull Request" y necesita pasar por diversas revisiones para su aprovación.

Cuando tu código sea aprovado pasará a un proceso de QA (Quality Assurance) donde se va a someter a diversas pruebas por varios usuarios.

Al terminar el ciclo, tu código pasará a producción y podrás ver tus nuevas actualizaciones en {{appName}} con tu nombre en la historia dentro del repositorio.
